# tcp-http-seasion

Per-method TCP HTTP session results (one file per method), plus helper scripts.

Files per method:
- `<method>.md`
- `<method>.json`

Scripts:
- `run_stress100_all_methods.sh`
- `summarize_stress100.py`

## Python per-method files

- `http_scapy.py`
- `http_tcpdump.py`
- `http_libpcap.py`
- `http_pypcap.py`
- `http_rawsocket.py`
- `http_rawsocket_tpacketv3.py`
- `http_dpkt.py`
- `http_ebpf.py`
- `http_tshark.py`
- `http_suricata.py`
- `http_netsniff.py`
- `http_zeek.py`
